
# Flask ToDo Manager

## Install

```bash
python -m venv venv
source venv/bin/activate      # Windows: venv\Scripts\activate
pip install -r requirements.txt
python app.py
```

Open http://127.0.0.1:5000

Features:
- Add Task
- Mark Done / Undo
- Delete Task
- Dashboard
- Bootstrap UI

Suggested classroom extensions:
1. Search
2. Filter by priority
3. Save to JSON
4. SQLite database
5. Login
6. REST API
