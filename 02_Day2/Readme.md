## IDE's for learning python 

IDE means integrated development environment, which is used as a editor similar to notepad++. It shows the source code in color, and when I do control + left mouse click it takes to the definition as well. 

In simple notepad if just for editing text file and IDE is for editing code. 

There are lot of IDE's available to work with python 
- Eclipse PyDev 
- VS Code (Visual studio code)
- pycharm 
- Juypter notebook
- IDLE (Which comes along with pyton installation)


### Which is the best IDE ?
- VS Code is the best IDE all over the developer community and its free and open source
- If you are working with the Data science, AI/ML then Juypter notebook is the best one. In this IDE python virtual machine in simple python is called as kernel (Don't get confused with this)

### Which one can I use to learn in this class
- Just use VS code (Visual Studio Code) from Microsoft 
- Install python and juypter extension in VS code that's it not more than that 


### Can I use command line to run the python code or vs code
- Until you learn python well use the CLI (Command Line Interface), in simple command prompt or terminal or power shell in Windows. Once you become expert you can use IDE to run. But I highly recommend to use CLI to execute python code 

### How to run python code in CLI 
```bash
python3 hello.py
```

### How to generate python byte code ? 
```bash
python3 -m compilealll hello.py 
```
This generate the bytecode in __pycache__ folder and bytecode in it
